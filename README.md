## Taiwan butterfly
#### 此APP收納了三種蝴蝶種類
```
粉蝶科、蛺蝶科、鳳蝶科
```
#### 每個科別各有五種蝴蝶，並且有他的詳細資訊
```
俗名、學名、食草、分佈地區、習性、可觀察之座標
```

<img src="https://github.com/weasel0630/mid_project/blob/master/mid/Assets.xcassets/butterfly%20class.png" >

